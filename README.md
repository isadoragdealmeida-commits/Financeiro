# 💰 FinFlow — Controle Financeiro Pessoal

App web de controle financeiro pessoal. Roda 100% no browser, sem backend, sem banco de dados — os dados ficam salvos no `localStorage` do seu navegador.

## Funcionalidades

- **Dashboard** — resumo completo: entradas, despesas, fatura do cartão, investimentos e saldo disponível
- **Entradas Fixas** — cadastre seus salários e renda fixa (salvos permanentemente)
- **Despesas** — registre por mês, com categoria e tipo (variável / fixa / cartão de crédito)
- **Investimentos** — registre aportes por mês
- **Metas** — defina metas de economia (ex: R$ 5.000, R$ 10.000) e acompanhe o progresso

## Como usar

Basta abrir o `index.html` no browser, ou fazer deploy na Vercel.

## Deploy na Vercel

1. Suba este projeto no GitHub
2. Acesse [vercel.com](https://vercel.com) → New Project
3. Importe o repositório
4. Clique em **Deploy** — pronto!

O `vercel.json` já está configurado.

## Tecnologias

- HTML5 + CSS3 + JavaScript puro (sem dependências)
- `localStorage` para persistência de dados
- Google Fonts (Syne + Space Mono)
- Responsivo para mobile e desktop
